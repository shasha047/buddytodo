# buddytodo

A Platform to create to-do list for buddy

## Getting Started

First you need to install all the dependency using, 
```
npm install
```
and run the system using 
```
nodemon
```
or
```
node server.js
```
 then open the browser and hit url "localhost:3000" and it will start

# Technology Used

* NodeJS
* MongoDB
* AngularJS
* Express
* Twitter-Bootstrap
* Javascript
* JWT
